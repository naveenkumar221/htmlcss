What is CSS?

CSS stands for Cascading Style Sheets . It is a stylesheet language used to describe the presentation of a document written in HTML.

CSS controls the layout of multiple web pages all at once. It allows you to apply styles to HTML elements, such as colors, fonts, spacing, and positioning.

CSS can be applied in three ways:

  • inline styles
  • internal stylesheets
  • external stylesheets

Inline styles: Applied directly to an HTML element using the style attribute.

CSS is essential for creating visually appealing and user-friendly web pages. It separates content from design, making it easier to maintain and update websites.

Selector

It is used to select an html element for styling 


        selector{
            property:value;
        }   

Example:
        h1{
            color:red;
        }

CSS Importance & Priority Order

When the same property is applied in all 3 types, the browser decides which one to keep based on specificity and !important. 

Default Priority (Highest → Lowest):

Inline CSS → wins over internal & external

Internal CSS → wins over external

External CSS → lowest by default

!important can change this order.

Example: 

           External CSS 
    <link rel="stylesheet" href="external.css">

           Internal CSS 
    <style>
        p {
            color: green !important; /* Overrides inline & external */
        }
    </style>

           Inline CSS 
    <p>  style="color: red;">This is a paragraph </p>
